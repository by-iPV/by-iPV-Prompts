# /prompts/chatgpt

## 1. Propósito
Repositorio de prompts para nodos ChatGPT orientados a gobernanza conceptual, TRC, peer review, análisis contextual y alineación ecosistémica.

## 2. Qué archivos contiene
- `README.md`
- `.gitkeep`
- (futuros prompts `.md` de uso ChatGPT)

## 3. Cuándo usarlo
Cuando se requiere diseño/validación conceptual, revisión por pares y consistencia narrativa del ecosistema.

## 4. Qué NO debe hacerse
No incluir runtime técnico pesado ni instrucciones de despliegue detallado.

## 5. Fuente primaria
ChatGPT.

## 6. Fuente secundaria
Codex.

## 7. Relación con MA-AE, N+1, HS v0.9.9, ECOSISTEMA MA-AE BD / HS / MM / PDGD, SDLC, Prompt Registry
Alinea MA-AE, preserva continuidad N+1, respeta freeze HS v0.9.9, habilita transición ECOSISTEMA MA-AE BD / HS / MM / PDGD y documenta contexto de soporte al SDLC dentro del Prompt Registry.

## 8. Riesgos si se usa incorrectamente
Sobre-cargarlo con detalle técnico puede generar desalineación entre intención conceptual y ejecución operativa.

## 9. Compatibilidad
- ChatGPT: alta
- Codex Cloud: media (consulta secundaria)
- VSCode: indirecta vía archivos versionados

## 10. Restricciones operacionales
No tocar runtime, UI ni deploy desde esta carpeta; uso documental.
